# Goliath

Applicazione desktop WPF (.NET) orientata alla gestione degli allenamenti in palestra.
L’obiettivo principale è fornire uno strumento per la creazione, gestione ed esecuzione di routine di allenamento in modo strutturato e intuitivo.

L’applicazione permette di:

Creare ed eliminare routine di allenamento personalizzate

Avviare ed eseguire sessioni di allenamento

Gestire un profilo utente, contenente le informazioni personali e le routine associate
